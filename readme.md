# worm
###### v1.0
(Ne)originální pokus o hru zvanou Snake, v tomto případě **Worm**.

---
##### Tenhle worm se umí:
- pohybovat pomocí šipek
- najíst
- zabít
- počítat si, kolik snědl
- hrát fajnou muziku (zkoušeno na Microsoft Edge, Chrome a Firefox jaksi nespolupracují)
